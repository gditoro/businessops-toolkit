This extension provides:
- Step-by-step Wizard (Intake)
- Save state files (`answers.yaml`, `company.yaml`)
- Run docs generator via CLI
- Copilot-friendly commands

## Development

1) Open repo in VS Code
2) `cd extensions/businessops-vscode`
3) `npm install` (optional; only needed if you add tooling)
4) `npm run compile`

Then press `F5` to open an Extension Development Host.

## Commands
- BusinessOps: Open Wizard (Intake)
- BusinessOps: Generate Docs
- BusinessOps: Open Docs Folder
