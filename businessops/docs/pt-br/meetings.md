# Cadência de Reuniões (Ritmo de Gestão)

<!-- BO:BEGIN GENERATED -->
## Princípios
- Reuniões existem para decisões, não para status.
- Toda reunião gera ações com donos e prazos.

## Cadência Semanal (ESSENCIAL)
### Revisão Semanal de Liderança (30–60 min)
**Participantes:** Fundador + Vendas + Operações + Financeiro
**Pauta:**
1. Caixa + Recebíveis
2. Pipeline de vendas + negócios em risco
3. Incidentes / gargalos operacionais
4. Riscos e bloqueios
5. Ações do roadmap para a semana

**Saída:** lista de ações + donos + prazos

## Cadência Mensal (RECOMENDADO)
### Revisão Mensal de Performance (60–90 min)
- Revisão de KPIs
- Revisão de margem
- Melhoria de processos
- Reclamações e clientes
- Pessoas/contratações

## Modelos
- Formato de ata:
  - Decisões
  - Ações
  - Riscos
  - KPIs impactados
